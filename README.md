## 📋 项目介绍

## 借款流程
1. 注册登录 开通资金托管
2. 前台申请额度 后台额度管理的额度列表进行审核通过
3. 前台个人主页-点击品质理财-个人借款-信用标（立即借款）
4. 后台初审管理-初审标（审核通过）
5. 前台智能投顾-投资列表（投标）

## 业务流程测试步骤
1. 注册并登录（判定）
2. 开户成功
  - 填写开户信息
  - 调用第三方开户系统
3. 额度申请
  - 提交额度申请
  - 管理员后台登录
  - 管理员进行审核通过
4. 发布借款成功
  - 提交借款申请
  - 管理员后台登录
  - 管理员审核借款申请
5. 借款流程图
